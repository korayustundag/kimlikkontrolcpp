# Security Policy

## Supported Versions

Son versiyonunda bir güvenlik açığı tespit edilmedi.

| Version   | Supported          |
| --------- | ------------------ |
| 1.0.2.0   | :white_check_mark: |
| 1.0.1.7   | :x:                |
| 1.0.0.0   | :x:                |

## Reporting a Vulnerability

Güvenlik açığı veya hafıza ile ilgili hataları açıklaması ile birlikte düzeltebilirsinz.
